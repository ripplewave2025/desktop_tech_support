# Manufacturer Support Resources

When Desktop Tech Support identifies issues that require manufacturer-specific tools, drivers, or hardware service, use these official resources. This guide covers the major Windows PC manufacturers.

---

## Dell

| Resource | URL |
|----------|-----|
| **Dell Support** | [support.dell.com](https://support.dell.com) |
| **SupportAssist** (auto-diagnostic tool) | [dell.com/support/contents/en-us/article/product-support/self-support-knowledgebase/software-and-downloads/supportassist](https://www.dell.com/support/home/en-us) |
| **Dell Drivers & Downloads** | [dell.com/support/home/en-us?app=drivers](https://www.dell.com/support/home/en-us?app=drivers) |
| **Dell Community Forums** | [dell.com/community](https://www.dell.com/community) |
| **Dell Service Tag Lookup** | Enter Service Tag at support.dell.com for model-specific support |
| **Dell ProSupport** (paid) | 24/7 expert support for business machines |

**Dell-specific tips:**
- Run `SupportAssist` for automated hardware diagnostics (pre-installed on most Dell machines)
- Use Service Tag (found on bottom of laptop or back of desktop) for exact driver matches
- Dell BIOS updates available through SupportAssist or manual download

## General Windows Resources

| Resource | URL | What it helps with |
|----------|-----|--------------------|
| **Microsoft Support** | [support.microsoft.com](https://support.microsoft.com) | Windows OS issues, updates, activation |
| **Windows Update Troubleshooter** | Built into Settings > Update & Security | Stuck updates |
| **DISM / SFC** | `DISM /Online /Cleanup-Image /RestoreHealth` then `sfc /scannow` | Corrupted system files |
| **Event Viewer** | `eventvwr.msc` | System error logs |
| **Reliability Monitor** | `perfmon /rel` | Crash history timeline |
| **Device Manager** | `devmgmt.msc` | Driver issues, hardware problems |
| **Disk Management** | `diskmgmt.msc` | Disk partitioning, drive letters |
| **Network Troubleshooter** | Settings > Network & Internet > Status | Connection issues |
| **Windows Memory Diagnostic** | `mdsched.exe` | RAM testing |
| **DirectX Diagnostic** | `dxdiag` | Graphics/audio driver info |

## How to Find Your Manufacturer

If you're not sure who made your computer, run this in PowerShell:

```powershell
Get-CimInstance Win32_ComputerSystem | Select-Object Manufacturer, Model
```

Or use this project's system info command:

```powershell
python -m cli.main --sysinfo
```
